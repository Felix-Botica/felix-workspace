## Lessons Learned
- **`openclaw doctor` is dangerous:** It auto-runs after every config patch/restart. It corrupted the Anthropic key on 27.03. Do NOT run it automatically or blindly.
- **Silent fails are unacceptable:** Health checks must test real API validity, not just "does the env var exist?"
- **LLM hallucinations on media:** Must explicitly instruct the model NOT to guess media content from placeholders
- **Gateway restarts kill sessions:** Never call `openclaw gateway restart` from within a session
- **Memory flush before compaction:** This pre-compaction flush system works well to preserve context
