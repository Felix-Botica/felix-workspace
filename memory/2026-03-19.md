## 🔴 Story Template — Mobile First Design Rules
80% der Shopify User sind mobile (und IG sowieso 99%)!
- Min font size: 28px (nichts darunter!)
- Alles bold/heavy weight
- Top safe zone: 250px (Status bar + "Deine Story" overlay)
- Bottom safe zone: 180px ("Schreibe etwas" + Swipe-Up)
- Badge NICHT in Ecke rechts oben (Close-Button!)
- Hoher Kontrast: Weiß auf Schwarz, grüner Discount-Preis
- Headline: 80px+, bold italic
- Produktname: 44px+, bold
- Preise: 52px+ für Discount, deutlich durchgestrichen für Original
- Code-Button: dicker Rahmen (4px+), 36px+ Text
- CTA: mit Background-Box, nicht nur floating Text
