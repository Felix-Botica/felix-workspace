### Story Published → Deleted → Published → Deleted
- First publish: no link parameter → no tappable link → Lothar deleted manually
- Second publish: with `link` parameter → API accepted silently but NO link sticker rendered → deleted again
- **Root cause:** Instagram Graph API does NOT support link stickers on Stories
- API silently ignores unknown parameters (no error, no warning) — misleading

### 🔴 Lesson: Instagram API Silent Parameter Acceptance
- Instagram's Graph API accepts ANY parameter without error
- "No error" ≠ "parameter is supported/working"
- Only the rendered output is source of truth
- Link stickers only available via: native app, Meta Business Suite, or Story Ads
- Same applies to: polls, quizzes, countdowns, music stickers
